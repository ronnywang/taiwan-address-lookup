# taiwan-address-lookup
有鑒於地址轉經緯度是一個很重要的需求，但是很少有可以離線大量查詢的版本，因此這個專案希望可以滿足這需求，讓大家可以更容易把門牌轉經緯度融合進自己的服務中

試用 http://ronnywang.github.io/taiwan-address-lookup/index.html
